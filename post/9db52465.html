Typora+PicGo+Gitee搭建图床上传环境

发表于 2023-10-26 更新于 2023-10-27 阅读次数： 本文字数： 382 阅读时长 ≈ 1 分钟

环境搭建前提

安装以下软件

  • Node.js
  • Git
  • PicGo

生成私人令牌

进入 gitee ，点击右上角个人资料头像，之后选择 账号设置，然后跳转后的左边栏点击 私人令牌 进行创建。

之后按照如下设置：

生成的个人访问令牌需要复制并保存好，因为关闭页面后将无法再次看到生成的令牌。

搭建 Gitee 图床服务

完成上述软件安装后，先打开 PicGo 的插件设置窗口，搜索 gitee 并安装 gitee-uploader。

之后打开图床设置选项卡，在弹出的下拉菜单中选择 gitee 图床，对其进行配置，填写 Gitee 相关信息。

其中

  • 图床配置名：可任意配置。
  • repo：配置图床仓库名，格式为 用户名/仓库路径。
  • branch：仓库分支名称。
  • token：前面生成的私人令牌。
  • path：仓库下的子文件夹。
  • 剩下的可不填写

填写完成后记得点击保存。

对 PicGo 进行配置，主要配置开机自启和，图片以时间戳重命名。

之后打开 Typora，点击菜单栏中的 文件，再选择 偏好设置，在在 Typora 的偏好设置中，点击 图像 选项卡进行上传图片的配置。

最后验证图片上传服务，点击验证图片上传选项进行验证，如果显示如下说明配置成功。
